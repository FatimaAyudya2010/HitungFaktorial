# Hitung Faktorial #

## Deskripsi Program
Program HitungFaktorial digunakan untuk menghitung faktorial dari sebuah bilangan bulat positif yang dimasukkan oleh pengguna.
Faktorial merupakan hasil perkalian berurutan dari 1 hingga angka tersebut (misalnya, 5! = 1 × 2 × 3 × 4 × 5 = 120).
Program ini menggunakan perulangan for untuk melakukan proses perhitungan.

## Cara Kerja Program
# Input Data dari Pengguna
-Program meminta pengguna untuk memasukkan angka bulat positif menggunakan kelas Scanner.
# Validasi Input
-Jika angka yang dimasukkan kurang dari 0, program akan menampilkan pesan:
angka harus positif!
-Jika angka valid (≥ 0), program melanjutkan ke proses perhitungan faktorial.
# Perhitungan Faktorial
-Program menggunakan perulangan for dari 1 hingga n.
-Setiap iterasi akan mengalikan nilai variabel faktorial dengan nilai i.
-Rumus umum:
 faktorial = 1 × 2 × 3 × ... × n
# Output Hasil
Program akan menampilkan hasil akhir berupa nilai faktorial dari angka yang dimasukkan.

## Contoh Input dan Output

# Input:
Masukkan angka bulat positif: 5
# Output:
Faktorial dari 5 adalah: 120
# Input:
Masukkan angka bulat positif: -3
# Output:
angka harus positif!
